# Bulls and Cows Game

C++ implementation of the classic logic game "Bulls and Cows"

This is a graphic version of the popular game where the player needs to guess the number guessed by the computer using the following clues: 
- Bull - the correct number in the correct position 
- Cow - the correct number in the wrong position

## Features
- Generate random 4-digit number without repeating digits
- Count bulls and cows for each attempt
- Check input for correctness (only numbers, no repeats)
- Support for different levels of complexity (number length)
- Pure C++ (STL, no external dependencies)
